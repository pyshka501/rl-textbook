# Translations

This directory contains translations of the textbook into other languages.

## Structure

Each language has its own directory named by language code:

```
translations/
├── ru/     # Russian (Русский)
├── zh/     # Chinese (中文)
├── es/     # Spanish (Español)
├── de/     # German (Deutsch)
├── fr/     # French (Français)
├── ja/     # Japanese (日本語)
└── ko/     # Korean (한국어)
```

## How to Translate

1. Copy the English chapter files from `book/chapters/` into your language directory
2. Translate the text while keeping all LaTeX commands, labels, and equations intact
3. Mathematical notation and variable names should remain in English/Latin
4. Update cross-references if chapter titles change
5. Submit a Pull Request with your translation
